# Acadexa AI

A modern AI-powered academic management platform that helps students manage assignments, notes, study plans, and AI-assisted learning in one place.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- FastAPI
- SQLAlchemy
- JWT Authentication

### Database
- PostgreSQL (Neon)

### AI
- Google Gemini API

## Features

- User Authentication
- Dashboard
- Assignment Manager
- Notes Manager
- AI Note Summarizer
- AI Quiz Generator
- Study Planner
- Calendar
- Notifications
